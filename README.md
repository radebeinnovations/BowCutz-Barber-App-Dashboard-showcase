# ✂️ BowCutz Barber App Dashboard (Showcase)

> **BowCutz** is a modern barber booking system built for a real barbershop.
> Clients book from their phones, and barbers manage everything from a live dashboard — no more WhatsApp bookings, no more confusion.

---

## 🚀 Features

* Real-time appointment booking
* Barber availability tracking
* Loyalty rewards system (free cut after 10 visits)
* Admin dashboard with live schedule and history
